# React-TypeScript-Vite-Boilerplate

This project serves as a base for all react vite typescipt projects

## Requirements

- Node >= 16.13
- NPM and PNPM

## Installation

- Download the repository
- Enable PNPM

```
corepack enable
```

- install dependencies

```
pnpm install
```

- build typescript code

```
pnpm build
```

- run the project

```
pnpm start
```
